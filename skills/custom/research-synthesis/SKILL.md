---
name: research-synthesis
description: Synthesize external sources into actionable project decisions, docs, or implementation guidance.
---
Use this skill for research-heavy work where literature, docs, or upstream methods must be turned into decisions.

Steps:
1. Gather primary sources.
2. Separate direct facts from inference.
3. Extract only what changes the project direction or implementation.
4. Write the result into project memory or a decision document.

Avoid dumping long notes without a concrete operational takeaway.
